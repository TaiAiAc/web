# 一些命令行工具

## @quiteer/scripts

一个命令行工具集，提供清理目录、依赖升级、规范化提交、发布流程、生成配置等能力。

- 可执行文件：`quiteer`、`qui`
- [《@quiteer/scripts 使用文档》](/cli/scripts)

## @quiteer/vite

一个命令行工具集，提供vite相关的功能，开发环境、打包构建等能力。

- 可执行文件：`qvite`
- [《@quiteer/vite 使用文档》](/cli/qvite)


## electronup

### 介绍

`electronup` 和 `create-electronup` 是一个用于 Electron 应用程序的发布工具，它可以帮助你将 Electron 应用程序打包成可执行文件。

### 文档地址

- [electronup 文档](https://quiteerjs.github.io/electronup/)
